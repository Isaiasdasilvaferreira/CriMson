1

Download e Instalação

Baixe o arquivo de setup da seção de download e execute o instalador. Siga as instruções na tela para completar a instalação.

2

Configuração Inicial

Após a instalação, execute o software pela primeira vez para configurar o perfil do seu sistema e preferências pessoais.

Otimização Automática

Use a ferramenta de scan automático para que o software identifique as melhores configurações para seu hardware.
